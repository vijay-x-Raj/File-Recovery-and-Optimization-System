File Access Mechanisms

Compare allocation methods and watch how blocks are accessed during read/write operations

Contiguous Allocation
Each file occupies a set of contiguous blocks on disk. The directory entry stores the start block and length.

Advantages

  • ✓ Fast sequential access
  • ✓ Simple implementation
  • ✓ Minimal seek time

Disadvantages

  • ✗ External fragmentation
  • ✗ File size must be known at creation
  • ✗ Difficult to grow files
